# GreenWallet

## API / Server Setup

### Install Flask (REST API)
```
pip install flask
pip install flask-cors
```

### Install Banking API
https://github.com/OpenBankProject/Hello-OBP-DirectLogin-Python

## App / Client Setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
